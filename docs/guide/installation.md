## 版本



## 使用包管理器

建议您使用包管理器 (NPM, Yarn, pnpm) 安装et-design, 然后您就可以使用打包工具，例如 Vite 和 webpack

#### NPM

```shell
$ npm install et-design --save
```

#### Yarn

```shell
$ yarn add et-design
```

#### pnpm
```shell
$ pnpm install et-design
```
